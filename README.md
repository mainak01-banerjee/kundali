# Kundali Generator & Astrology Engine

A Spring Boot based astrology engine that generates accurate birth charts (Kundali), planetary positions, Nakshatras, Dashas, and visual chart representations using Swiss Ephemeris.

---

## 🚀 Features

- Generate Birth Horoscope (Lagna Chart)
- Accurate Planetary Positions (Swiss Ephemeris)
- Nakshatra & Pada Calculation
- House Mapping
- Vimshottari Mahadasha & Antardasha
- North Indian Chart (SVG)
- East Indian Chart (SVG)

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Maven
- Swiss Ephemeris
